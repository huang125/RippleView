# RippleView
A view similar ripple effect.
## How do I use it?
### Stup
#### Gradle
```
dependencies {
  // jCenter
  compile 'com.huang.rippleview:rippleview:0.0.1'
}
```
#### Maven
```
<dependency>
  <groupId>com.huang.rippleview</groupId>
  <artifactId>rippleview</artifactId>
  <version>0.0.1</version>
  <type>pom</type>
</dependency>
```
